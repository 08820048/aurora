## 相关技术

### 前端

前台：vue3 + pinia + vue-router + axios + tailwindcss + element-ui

后台：vue2 + vuex + vue-router + axios + element-ui

### 后端

springboot + springsecurity + mysql + redis + mybatisplus + quartz + rabbitmq + maxwell + elasticsearch + minio + docker + nginx
